# НашеНачало - Платформа для поиска соседей по аренде жилья

Существует проблема поиска жилья на рынке аренды для совместного проживания . Наше Приложение базируется на: расширенной анкете пользователя,
Поиск соседа по интересам,
Разделение арендной платы.
Описанное выше поможет молодым людям находить  доступное жилье в предпочтительном районе/городе, решается вопрос возможного одиночества при съёме недвижимости и убирает факт высокой аренды.
При разработки были задействованы следующие средства: ИИ инструменты – гроке, Алиса АИ, гид зал копилейт; верцел – облачная платформа для разворачивания,  масштабирования и поддержки веб приложений.

## 🛠️ Технологический стек
- **HTML**
- **TypeScript**
- **Tailwind CSS**
